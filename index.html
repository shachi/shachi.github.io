LoopRoute Privacy Policy

Last updated: 2026-03-08

This app ("LoopRoute") records walking/running routes and workout metrics.

Data We Use

  • Location data: Used to generate loop routes and record distance/time during sessions.
  • Health data (optional): Used to read/write workout data via Apple Health when the user enables Health integration.
  • iCloud/CloudKit data: Used to sync app data (history and saved waypoints) across the user's own devices.

Why We Use Data

  • Route generation from current location.
  • Session recording (time, distance, pace).
  • Workout save/read integration with Apple Health (if enabled by user).
  • Backup/sync via iCloud.

Data Sharing

  • We do not sell user data.
  • We do not share personal data with third-party advertisers.
  • Data is stored on-device and in the user's iCloud account (CloudKit), controlled by Apple platform permissions.

Data Retention

  • Session history and saved waypoints are stored until the user deletes them.
  • Users can delete data in-app (history and waypoints), and can disable permissions in iOS/watchOS Settings at any time.

Permissions

  • Location permission (When In Use / Always) is requested only for route generation and background session recording.
  • Health permission is requested only for workout integration.

LoopRoute プライバシーポリシー

最終更新日: 2026-03-08

本アプリ（「LoopRoute」）は、ウォーキング/ランニングのルート記録およびワークアウト連携機能を提供します。

利用する情報

  • 位置情報: 周回ルート生成、計測中の距離/時間記録に使用します。
  • ヘルスケア情報（任意）: ユーザーが許可した場合のみ、Appleヘルスケアのワークアウト情報の読み書きに使用します。
  • iCloud/CloudKit データ: 履歴および保存済み寄り道ポイントの、同一ユーザー端末間同期に使用します。

利用目的

  • 現在地を起点としたルート生成
  • セッション中の時間・距離・ペースの記録
  • Appleヘルスケアとの連携（許可時）
  • iCloudバックアップ/同期

第三者提供

  • ユーザーデータの販売は行いません。
  • 広告目的で第三者へ個人情報を提供しません。
  • データは端末内およびユーザー自身の iCloud（CloudKit）に保存され、Appleの権限管理に従います。

保存期間

  • 履歴と寄り道ポイントは、ユーザーが削除するまで保存されます。
  • ユーザーはアプリ内削除機能、またはOS設定から権限停止をいつでも行えます。

権限について

  • 位置情報（使用中/常時）は、ルート生成とバックグラウンド計測のためにのみ利用します。
  • ヘルスケア権限は、ワークアウト連携のためにのみ利用します。
